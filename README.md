# Autosprowadzone Redesign

This project is a redesign of the original 'Autosprowadzone' website, focusing on modern web development practices, including:

- **SEO Optimization**: Ensuring high visibility in search engine results.
- **Mobile-First Design**: Prioritizing mobile user experience.
- **Performance**: Fast loading times and smooth interactions.
- **Latest Technologies**: Utilizing Next.js, React, and Tailwind CSS.

## Getting Started

To run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

The project maintains a similar page structure to the original, with improvements in component design and overall implementation.
